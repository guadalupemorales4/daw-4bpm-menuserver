La historia de HTML

Empieza en 1989 cuando el investigador
Tim Berners- Lee lo inventa con el fin de compartir artículos científicos.
La primera versión sólo contaba con 18 etiquetas para texto, podía ligar varios documentos con el hipertexto.
Todo esto se podía visualizar en un navegador llamado World Wide Web.

En 1993 y 1996
  • En 1993, un organismo que determinó las reglas para el HTML.
  • En 1996 llegó la versión HTML 3.2
  • una de las más completas
  • hasta nuestros días.

En la tercera versión de HTML
se incluyen las etiquetas específicas
para los navegadores web como Internet Explorer
pero no funciono por lo que surgio la version 4.0
en la cual se agregaron los botones
y el iFrame para poder tener sitios dentro de otros sitios
